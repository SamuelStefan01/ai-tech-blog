## AI & Tech Blog

is a modern single-page web application built with vanilla HTML, CSS, and JavaScript.
The project focuses on clean architecture, dynamic client-side routing, REST API integration, responsive design, and advanced UI customization including theme and color-palette switching.
Articles are loaded dynamically from a remote REST API (WT server), while the UI supports dark/light mode, custom color palettes, Google authentication, and a mobile-friendly off-canvas navigation.
The goal of the project is to demonstrate practical frontend development skills without frameworks, emphasizing clarity, maintainability, and real-world web standards.

## Tech Stack
- HTML5
- CSS3 (custom properties, responsive design)
- Vanilla JavaScript (SPA routing, state handling)
- REST API integration (WT server)
- Google OAuth Sign-In
- GitHub Pages deployment
